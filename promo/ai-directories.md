### 通用提交信息

**Tool Name:** EFC (Everything Fact-Checked)

**Category:** Research / Content Verification / AI Safety

**One-line description:** Catches the 5 systematic errors in AI-generated research reports — unit errors, fabricated data, source conflation, stale figures, and attribution laundering.

**Full description:**
EFC is a fact-checking tool that catches systematic errors in AI-generated research reports and data-heavy documents. It identifies 5 repeatable failure modes: unit/scale errors ($4.2B → $4,200B), fabricated interpolation (charts with made-up data points), source conflation (GMV cited as revenue), stale data presented as current, and attribution laundering (blogs cited as official filings).

Available as a CLI tool (pip install), GitHub Action (auto-check reports in PRs), and a standalone agent skill (one Markdown file, works with any AI agent). The source-content verification feature fetches cited URLs and checks whether claimed figures actually appear in the source text. Stdlib-only Python, MIT licensed, zero dependencies.

**Pricing:** Free / Open Source (MIT)
